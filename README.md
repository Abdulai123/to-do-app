# To Do Application Using Laravel

## About Laravel

This is a To Do List application written in laravel it's very simple and easy to use with cool UX/UI

  ### Functionalities
- Add to do task
- Edit to do task
- Update to do task
- Delete to do task

## Installation
- composer install
- php artisan migrate
- php artisan serve go to your browser and open 127.0.0.1:8000

## Requirements
- PHP
- composer
- mysql
